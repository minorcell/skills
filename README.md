# minorcell-skills

A collection of skills distilled from minorcell's own workflows and useful practices from others, shared for use with AI Agents (Claude Code, Codex, etc.).

## Prerequisites

- Node.js environment installed
- Ability to run `npx skills` commands

## Installation

> **Tip**: Install only the skills you actually need. Each installed skill adds context for your AI agent to consider.

### Quick Install

```bash
npx skills add minorcell/skills
```

## Available Skills

| Name                     | Description                                                                                                                                                                             |
| ------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `extract-video-content`  | Helps extract verifiable evidence from public Bilibili and YouTube videos, including metadata, subtitles, comments, audio, and key frames, for accurate summaries, analysis, and research. |
| `writing-blog`           | Helps write, improve, and review technical or general blog posts using proven writing practices from experienced practitioners.                                                |
| `writing-proposals`      | Helps draft, rewrite, review, and standardize technical proposals, RFCs, design issues, API proposals, tooling proposals, and implementation plans.                            |
| `writing-product-docs`   | Helps draft, rewrite, and review product definition documents that explain the product, problem, users, core concepts, and information structure.                              |
| `web-design`             | Helps design, implement, review, and improve high-quality web interfaces, UI components, design systems, and layouts.                                                          |
| `web-performance`        | Helps audit and optimize web performance through code-level, locally actionable improvements to loading, JavaScript execution, rendering stability, and framework usage.       |
